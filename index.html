Private Key Generator

This key derivation function allows you to generate a private key for the Secp256k1 curve using any seed and password of choice. By providing your own seed and password, you can create a unique and memorable seed for your private key.

This is a client-side script, meaning it can be executed offline by disconnecting from internet once the page has loaded. The source code is available at: GitHub Repository.

Algorithm

  1. Encode both seed and password to UTF-8.
  2. Hash (keccak-256) the encoded seed and password separately r amount of times, where r = seed.length * password.length 2.
  3. Concatenate the hashed seed with the hashed password and hash the concatenated hashes for r rounds again.
  4. Calculate the private key (d) by bringing the resulting hash within the interval 1 to n (order of curve). 
    hash mod n = d
if d == 0
    request new input
else
    return d
